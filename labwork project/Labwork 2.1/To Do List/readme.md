# 🛍️ Product Display Card

## 📖 Overview
This project is a simple **Product Display Card Layout** created using **HTML5** and **CSS3**. It displays multiple air conditioner products in a clean card-based layout with product images, descriptions, prices, discounts, and action buttons.

## ✨ Features
- ✅ Responsive card-based product layout
- 🖼️ Product image section
- 📝 Product name and description
- 💰 Price and MRP display
- 🎉 Discount percentage
- 🛒 "Add to Cart" and "See Option" buttons
- 🎨 Clean and modern UI using CSS

## 💻 Technologies Used
- 🌐 HTML5
- 🎨 CSS3

## 🚀 How to Run

1. 📥 Download or clone the project.
2. 📂 Open the project folder.
3. ✅ Make sure the **css** and **images** folders are in the correct locations.
4. 🌐 Open `index.html` in your web browser.

## 🛍️ Layout Description

Each product card contains:
- 🖼️ Product Image
- 📝 Product Description
- 💰 Current Price
- 💲 Original MRP
- 🎉 Discount Information
- 🛒 Action Button

The page displays five different AC products:
- ❄️ Tata Vectra Elite
- ❄️ Hitachi Split AC
- ❄️ Voltas Vectra Elite
- ❄️ Cruise Split AC
- ❄️ Daikin Split AC

## ⚙️ Customization

You can easily customize the project by:
- 🖼️ Changing product images inside the **images** folder.
- ✍️ Updating product names and descriptions in `index.html`.
- 🎨 Modifying colors, fonts, spacing, and card styles in `style.css`.
- ➕ Adding more product cards by copying the existing card structure.
