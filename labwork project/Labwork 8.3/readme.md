# Team Profile Card UI

A simple Team Profile Card project built using **HTML5** and **CSS3**. This project demonstrates the use of the CSS Box Model, inline-block layout, borders, margins, padding, background images, Font Awesome icons, and Google Fonts.

## 📌 Features

- Responsive card-style layout
- Team member profile section
- Profile image with border styling
- Social media icons
- Custom Google Fonts
- Font Awesome icons
- CSS Box Model implementation
- Clean and simple user interface

## 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome

1. Download or clone the project.

```

2. Open the project folder.

3. Double-click **index.html** or open it using **Live Server** in Visual Studio Code.
