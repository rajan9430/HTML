# 🛒 Flipkart Clone Website

A simple **Flipkart Clone Website** developed using **HTML5**. This project recreates the basic layout and design of the Flipkart website, including the Home Page, Login Page, and Sign Up Page. It is built for learning HTML structure and webpage design without using any CSS frameworks or JavaScript.

---

## 📌 Project Overview

This project demonstrates how to create a multi-page e-commerce website using pure HTML. It includes a navigation bar, product categories, search bar, login page, sign-up page, and footer similar to Flipkart.

The project is suitable for beginners who want to practice HTML and understand how webpages are structured.

---

- 🏠 Flipkart Home Page

## 🚀 How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/rajan9430/HTML.git
```

2. Go to the Lab project folder.

2. Open the flipkart project folder.

3. Double-click **index.html** or open it in any web browser.
